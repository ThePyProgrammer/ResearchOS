  <action>
1. **Add import models to `backend/models/experiment.py`:**
   ```python
   class ExperimentImportItem(CamelModel):
       tmp_id: str
       parent_tmp_id: Optional[str] = None
       name: str
       config: dict[str, Any] = {}
       metrics: dict[str, Any] = {}
       collision_action: Literal["create", "update", "skip"] = "create"
       existing_id: Optional[str] = None

   class ExperimentImportRequest(CamelModel):
       items: list[ExperimentImportItem]  # BFS-ordered, parents first
       parent_id: Optional[str] = None    # root target group
       merge_metrics: bool = False

   class ExperimentImportResult(CamelModel):
       tmp_id: str
       status: Literal["created", "updated", "skipped"]
       id: Optional[str] = None
   ```

2. **Add `bulk_create_experiment_tree()` to `backend/services/experiment_service.py`:**
   - Accepts project_id, items (BFS-ordered list), parent_id (root target), merge_metrics flag
   - Iterates items in order, building a `tmp_id -> real_id` map as experiments are created
   - For each item:
     - Resolve real parent_id: `id_map[item.parent_tmp_id]` if parent_tmp_id set, else the root parent_id
     - If collision_action == "skip": append skipped result, continue
     - If collision_action == "update" and existing_id set: merge or overwrite metrics on existing experiment using update_experiment, map tmp_id to existing_id
     - If collision_action == "create": call create_experiment with resolved parent_id, map tmp_id to new id
   - Return list of ExperimentImportResult

3. **Add endpoint to `backend/routers/experiments.py`:**
   ```python
   @router.post("/api/projects/{project_id}/experiments/import-csv", status_code=201)
   def import_csv(project_id: str, req: ExperimentImportRequest):
       results = bulk_create_experiment_tree(
           project_id=project_id,
           items=req.items,
           parent_id=req.parent_id,
           merge_metrics=req.merge_metrics,
       )
       return [r.model_dump(by_alias=True) for r in results]
   ```

4. **Add `importCsv` to frontend API client** in `frontend/src/services/api.js`:
   ```javascript
   // Add to experimentsApi object:
   importCsv: (projectId, data) =>
     apiFetch(`/projects/${projectId}/experiments/import-csv`, { method: 'POST', body: data }),
   ```
