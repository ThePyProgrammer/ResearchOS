# Phase 9 Plan 02: Kanban Board with DnD Card Movement

Full Kanban board replacing the placeholder, with @dnd-kit drag-and-drop for both card movement between columns and column reordering, inline task creation per column, column management (rename, add, delete with task migration, reorder, color), and all required exported helper functions for unit testing.

## What was built

The `KanbanView` component replaces `KanbanPlaceholder` in `ProjectTasks.jsx`. The implementation uses two nested `DndContext` instances with separate IDs to avoid sensor collision (per research pitfall 3): `task-column-dnd` for column reordering and `task-card-dnd` for card movement between/within columns.

Additionally, the linter had pre-populated the file with `isOverdue`, `applyTaskFilter`, `sortTaskRows`, `getMonthGrid` exported functions and a full `TaskListView` component. The Kanban implementation built on top of that foundation.
