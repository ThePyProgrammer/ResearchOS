<task type="auto">
  <name>Task 1: CitationNetworkViz d3 force graph component</name>
  <files>frontend/src/components/CitationNetworkViz.jsx</files>
  <action>
    Create `CitationNetworkViz.jsx` following the NoteGraphView.jsx pattern closely. The component:

    **Props:** `{ papers, colorBy, sizeBy, showAuthorEdges, showVenueEdges, projectId }`

    **Refs:**
    - `svgRef` — target for d3.select
    - `svgWrapperRef` — for container size measurement
    - `simRef` — d3 simulation reference for cleanup

    **State:**
    - `hoveredNode` — null or { paper, x, y } for tooltip positioning

    **Core useEffect (buildGraph)** — triggered when papers, showAuthorEdges, showVenueEdges change:
    1. Clear existing SVG content: `d3.select(svgRef.current).selectAll('*').remove()`
    2. Measure container: `const { width, height } = svgWrapperRef.current.getBoundingClientRect()`
    3. Build edges: `const { authorEdges, venueEdges } = buildCitationEdges(papers)` (import from ProjectReviewDashboard)
    4. Filter edges by toggle state: combine `showAuthorEdges ? authorEdges : []` and `showVenueEdges ? venueEdges : []` into `activeEdges`
    5. Build degree map from activeEdges for node sizing
    6. Create nodes array: `papers.map(p => ({ ...p, id: p.id }))`
    7. Create SVG groups: `const g = svg.append('g')` (zoom target)
    8. Draw links:
       - Author edges: blue (`#3b82f6`), solid stroke, opacity 0.3
       - Venue edges: orange (`#f97316`), dashed stroke (`strokeDasharray: '5,3'`), opacity 0.3
    9. Draw nodes as circles:
       - Fill: computed via `getNodeColor(paper, colorBy, scales)` — build scales from paper data (d3.scaleSequential for year, categorical map for venue)
       - Radius: computed via `getNodeSize(paper, sizeBy, degreeMap)`
       - Stroke: white 1.5px for contrast
       - Cursor: pointer
    10. Add text labels below each node: paper title truncated to 20 chars, font-size 9px, text-anchor middle
    11. Set up d3.forceSimulation:
        - `forceLink(activeEdges).id(d => d.id).distance(100).strength(0.6)`
        - `forceManyBody().strength(-250)`
        - `forceCenter(width/2, height/2)`
        - `forceCollide(d => getNodeSize(d, sizeBy, degreeMap) + 15)`
    12. On tick: update link x1/y1/x2/y2 and node transform positions
    13. Set up d3.zoom() with scaleExtent [0.3, 4]: `svg.call(d3.zoom().on('zoom', (e) => g.attr('transform', e.transform)))`
    14. On simulation end: call fitView (zoom to fit all nodes in viewport with padding)
    15. Mouse events on nodes:
        - mouseenter: setHoveredNode with paper data + screen position
        - mouseleave: setHoveredNode(null)
        - click: `window.location.href = /library/paper/${d.id}` (or `/library/website/${d.id}` if itemType === 'website')
    16. Cleanup: return `() => { if (simRef.current) simRef.current.stop() }`

    **Separate useEffect for color/size updates** — triggered when colorBy, sizeBy change but NOT requiring simulation rebuild:
    - Re-compute scales from papers
    - Update existing node circle `fill` and `r` attributes via d3.selectAll
    - Do NOT restart simulation — just update visual properties

    **fitView function:** Same as NoteGraphView — compute bounding box of all nodes, calculate scale to fit within container with 40px padding, apply zoom transform.

    **Tooltip JSX** (below SVG):
    ```jsx
    {hoveredNode && (
      <div className="absolute z-50 bg-slate-800 text-white text-[11px] rounded-lg px-3 py-2 pointer-events-none shadow-lg max-w-xs"
        style={{ bottom: 12, left: 12 }}>
        <div className="font-semibold truncate">{hoveredNode.title}</div>
        <div className="text-slate-300 mt-0.5">{hoveredNode.authors?.slice(0,3).join(', ')}{hoveredNode.authors?.length > 3 ? ' et al.' : ''}</div>
        <div className="text-slate-400 mt-0.5">{hoveredNode.venue} {hoveredNode.year}</div>
      </div>
    )}
    ```

    **Container:** Relative positioned div with ref={svgWrapperRef}, height ~60vh (or min 400px), SVG fills 100% width and height.

    **Edge case handling:**
    - 0 papers: show empty state message "No papers linked to this project"
    - 1 paper: show single node centered, no edges
    - Papers with missing year/venue: use fallback values in color/size functions
  </action>
