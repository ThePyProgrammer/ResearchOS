  <action>
    Write tests FIRST in ProjectReviewDashboard.test.jsx importing the named exports from ProjectReviewDashboard.jsx. Tests should be pure unit tests — no React rendering, no d3, no DOM.

    Then create ProjectReviewDashboard.jsx with the following exported pure functions:

    1. `normalizeAuthor(name)` — lowercase, strip punctuation, split on whitespace. For "Last, First" format: return `last f`. For "First Last" format: return `last f`. For single-name: return the name lowercased. This is the critical function per Pitfall 1 in RESEARCH.md.

    2. `buildCitationEdges(papers)` — O(n^2) comparison. For each paper pair, check:
       - Shared authors: normalize each author, find intersection. If >= 1 shared → add to authorEdges array as { source: paper1.id, target: paper2.id, type: 'author', sharedAuthors: [...] }
       - Same venue: case-insensitive venue match (both non-empty) → add to venueEdges as { source, target, type: 'venue' }
       Returns { authorEdges, venueEdges }.

    3. `getNodeColor(paper, colorBy, scales)` — where scales is { yearScale, venueColorMap, typeColorMap }. Switch on colorBy:
       - 'year': return scales.yearScale(paper.year)
       - 'venue': return scales.venueColorMap[paper.venue] || '#94a3b8'
       - 'type': return paper.itemType === 'website' ? '#14b8a6' : '#3b82f6'
       - 'uniform': return '#3b82f6'

    4. `getNodeSize(paper, sizeBy, degreeMap)` — Switch on sizeBy:
       - 'connections': return Math.max(6, Math.min(20, 6 + (degreeMap[paper.id] || 0) * 2))
       - 'year': return Math.max(6, Math.min(20, 6 + Math.max(0, (paper.year - 1990) / 3)))
       - 'uniform': return 8

    5. `computeTimelinePositions(papers)` — group by year, assign _y = index within group. Return papers.map with added _x (year or 0) and _y fields.

    6. `buildHeatmapMatrix(papers, rowAxis, colAxis)` — For each paper, extract dimension values for rowAxis and colAxis. Dimensions: 'tags' → paper.tags (array, explode), 'venue' → [paper.venue], 'year' → [String(paper.year)], 'author' → paper.authors (array, explode but use first 3 only to avoid matrix explosion). Build a count map keyed by `${row}||${col}`. Handle empty tags/venue as "(no tags)"/"(no venue)". Return { rows: string[], cols: string[], cells: Array<{row, col, count}> }.
