From frontend/src/pages/ProjectNotesIDE.jsx (the DOM walker pattern):
```javascript
// _nodeToMd walks HTML DOM tree and maps each element to markdown.
// The LaTeX serializer follows the SAME pattern but maps to LaTeX.
function _nodeToMd(node) {
  if (node.nodeType === Node.TEXT_NODE) return node.textContent
  if (node.nodeType !== Node.ELEMENT_NODE) return ''
  const tag = node.tagName.toLowerCase()
  const inner = () => Array.from(node.childNodes).map(_nodeToMd).join('')
  switch (tag) {
    case 'h1': return `# ${inner()}\n\n`
    // ... etc
    case 'span':
      if (node.dataset.wikiName) return `[[${node.dataset.wikiName}]]`
      if (node.dataset.latex)    return `$${node.dataset.latex}$`
      return inner()
    case 'table': // handles tr/th/td
  }
}
function htmlToMarkdown(html) {
  const doc = new DOMParser().parseFromString(html, 'text/html')
  return _nodeToMd(doc.body).trim()
}
```

def _make_citation_key(paper) -> str:
    last_name = "unknown"
    if paper.authors:
        first_author = paper.authors[0]
        if "," in first_author:
            last_name = first_author.split(",")[0].strip()
        else:
            tokens = first_author.split()
            last_name = tokens[-1] if tokens else "unknown"
    last_name = re.sub(r"[^a-zA-Z]", "", last_name).lower()

    title_word = ""
    for word in paper.title.split():
        clean = re.sub(r"[^a-zA-Z]", "", word).lower()

    Create `frontend/src/utils/latexSerializer.js` exporting two functions:

    1. `escapeLatex(text)` — escapes LaTeX special characters: `\` to `\textbackslash{}`, `{` to `\{`, `}` to `\}`, `&` to `\&`, `%` to `\%`, `#` to `\#`, `$` to `\$`, `^` to `\textasciicircum{}`, `_` to `\_`, `~` to `\textasciitilde{}`. Order matters: backslash first.

    2. `htmlToLatex(html, options)` — public function. Uses `DOMParser` to parse HTML, walks the DOM tree with a recursive `_nodeToLatex(node, ctx)` function (exactly parallel to `_nodeToMd` in ProjectNotesIDE.jsx). The `ctx` object carries `{ usedKeys: new Set() }` to track cited keys.

    DOM walker mapping (switch on `tag`):
    - `h1/h2/h3` -> `\section{}/\subsection{}/\subsubsection{}`
    - `p` -> `inner()\n\n`
    - `strong/b` -> `\textbf{}`
    - `em/i` -> `\textit{}`
    - `u` -> `\underline{}`
    - `s` -> `\sout{}` (requires \usepackage{ulem})
    - `code` (not inside pre) -> `\texttt{}`
    - `pre` -> `\begin{verbatim}...\end{verbatim}`
    - `blockquote` -> `\begin{quote}...\end{quote}`
    - `hr` -> `\hrule`
    - `ul/ol` -> `\begin{itemize}/\begin{enumerate}` with `\item` children
    - `li` -> `  \item inner()`
    - `table` -> call `_serializeTable(tableEl, ctx)` helper
    - `span` with `data-cite-key` -> `\cite{key}`, add key to `ctx.usedKeys`
    - `span` with `data-latex` -> `$${node.dataset.latex}$` (NO escaping)
    - `span` with `data-wiki-name` -> just emit inner text (wiki links are not LaTeX-meaningful)
    - `a` -> `\href{url}{text}` (requires \usepackage{hyperref})
    - Text nodes -> `escapeLatex(node.textContent)` EXCEPT when inside a math context

    `_serializeTable(tableEl, ctx)`:
    - Get all `tr` rows
    - Count max columns from first row's th/td
    - Build `\begin{tabular}{| l | l | ... |}` column spec
    - First row as header with `\hline` after
    - Body rows with `&` separators and `\\` line endings
    - Close with `\end{tabular}`

    Return value of `htmlToLatex`: `{ latex: string, usedKeys: Set<string> }`

    CRITICAL: Text nodes inside `<span data-latex>` must NOT be escaped — the content is already LaTeX math.
